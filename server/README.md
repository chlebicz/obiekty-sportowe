# Server
Built using NestJS

## Project setup

```bash
# install necessary dependencies
$ npm install
```

You will also need to provide a configuration file. Create a `config.json` file based on `config-example.json` and fill in your data.

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Run the scraper

```bash
$ npm run scrape
```
